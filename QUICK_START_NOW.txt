🆘 IF SOMETHING DOESN'T WORK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BLANK PAGE?
  → Open browser console: F12
  → Look for red error messages
  → Read the error and check documentation

DATABASE ERROR?
  → Make sure you loaded supabase-schema.sql successfully
  → Go back to Step 1 and verify "Success" message appeared

CAN'T SEE BEAUTIFUL DESIGN?
  → Rebuild with: npm run build
  → Restart dev server: npm run dev
  → Clear browser cache: Ctrl+Shift+Delete

APP WON'T START?
  → Run: npm install
  → Then: npm run dev

ERROR ON LOGIN?
  → Make sure you created a test user in Supabase
  → Make sure user is assigned to a company
  → Check browser console for error details

GOOGLE BUTTON DOESN'T WORK?
  → That's normal for now - it needs Google OAuth setup
  → See: GOOGLE_OAUTH_TROUBLESHOOTING.md for setup
