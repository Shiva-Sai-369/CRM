## 🎨 Design System

All colors use Tailwind CSS:
- `from-indigo-600` / `to-purple-600` - Gradients
- `bg-indigo-50` / `bg-indigo-100` - Backgrounds
- `text-indigo-600` / `text-indigo-700` - Text
- `border-gray-300` - Borders
- `rounded-lg` / `rounded-3xl` - Border radius
- `shadow-lg` / `shadow-xl` - Shadows

## 🔧 Technical Details

### Technologies Used
- **Tailwind CSS** - Styling framework
- **React Hooks** - State management
- **SVG Icons** - All icons are SVG
- **Gradients** - CSS gradients for backgrounds
- **Responsive** - Mobile-first design

### Key Classes
- `bg-gradient-to-br` - Gradient background
- `rounded-3xl` - Large border radius for modern look
- `shadow-2xl` - Large shadow for depth
- `transition-all` - Smooth animations
- `focus:ring-indigo-500` - Focus states
- `disabled:opacity-50` - Disabled states
